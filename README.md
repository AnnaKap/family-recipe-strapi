# Getting started with family recipe strapi

this is the strapi CMS for the family recipe app (WIP)

to run copy the .env.example and set up your environment variables

## google OAuth
this app is configured to run with google OAuth for non-admin users
to have this set up correctly you must create an oauth app on google console and get your variables (first you will need to start and ngrok tunnel please keep readin below)

you must also run the application using an ngrok tunnel. You can run the command

```
npm run start-with-ngrok
```

it will start strapi as well as an ngrok tunnel that strapi will use as its url in the /config/server.ts file

with please use this ngrok tunnel url to configure your google Oauth on the google console

# üöÄ Getting started with Strapi

Strapi comes with a full featured [Command Line Interface](https://docs.strapi.io/dev-docs/cli) (CLI) which lets you scaffold and manage your project in seconds.

### `develop`

Start your Strapi application with autoReload enabled. [Learn more](https://docs.strapi.io/dev-docs/cli#strapi-develop)

```
npm run develop
# or
yarn develop
```

### `start`

Start your Strapi application with autoReload disabled. [Learn more](https://docs.strapi.io/dev-docs/cli#strapi-start)

```
npm run start
# or
yarn start
```

### `build`

Build your admin panel. [Learn more](https://docs.strapi.io/dev-docs/cli#strapi-build)

```
npm run build
# or
yarn build
```
